1. Get-Help
Menampilkan dokumentasi dan bantuan untuk cmdlet tertentu.

2. Get-Command
Menampilkan daftar cmdlet, fungsi, dan alias yang tersedia.

3. Get-Process
Menampilkan daftar proses yang sedang berjalan.

4. Stop-Process
Menghentikan proses yang sedang berjalan.

5. Get-Service
Menampilkan daftar service pada sistem.

6. Start-Service
Menjalankan service tertentu.


7. Stop-Service
Menghentikan service tertentu.

8. Get-ChildItem
Menampilkan isi direktori sama dengan ls

9. Set-Location
Berpindah direktori (setara dengan cd).

10. New-Item
Membuat file atau folder baru.

11. Remove-Item
Menghapus file atau folder.

12. Copy-Item
Menyalin file atau folder.

13. Move-Item
Memindahkan file atau folder.

14. Rename-Item
Mengganti nama file atau folder.


15. Get-Content
Menampilkan isi file teks.

16. Set-Content
Menulis atau mengganti isi file.

17. Add-Content
Menambahkan isi ke file tanpa menghapus isi lama.

18. Clear-Host
Membersihkan tampilan terminal.

19. Get-Date
Menampilkan tanggal dan waktu saat ini.

20. Test-Path
Mengecek apakah file atau folder ada.
